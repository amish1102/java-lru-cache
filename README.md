# Java LRU Cache

A simple implementation of an LRU (Least Recently Used) Cache in Java using:

- HashMap
- Doubly Linked List

## Features

- O(1) `get()`
- O(1) `put()`
- Manual Doubly Linked List implementation
- Dummy head/tail node optimization
